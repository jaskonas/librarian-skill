# Mode: Audit

Goal: report integrity problems across the library and write them to a health report. Audit
**only reports** — it never deletes or rewrites notes or `.bib` entries. The human decides
what to fix.

Read `references/conventions.md` and `references/provenance.md` first. Load
`Library/config.md` for `books_folder`, `bib_path`, `vault_name`.

## Checks

### 1. Validation / lint

For every `type: book-note`:

- Required fields present and non-empty: `title`, `authors`, `year`, `citekey`.
- ISBN (if present) passes validation: `python scripts/bibtools.py check-isbn <isbn>`.
- `status` is one of exactly: `to-read`, `reading`, `read`, `abandoned`.
- `year` is a plausible 3–4 digit number.

### 2. Duplicate detection

- The same `isbn` appearing on more than one note.
- The same normalized `title` + first author across more than one note.
- Duplicate `citekey`s (the same key on two notes, or repeated in the `.bib`).

### 3. Orphan reconciliation

Parse the `.bib`:

```bash
python scripts/bibtools.py parse "<bib_path>"
```

- `.bib` entries whose `citekey` matches **no** note.
- Book notes whose `citekey` matches **no** `.bib` entry (or that have no citekey at all).

**Report only — never delete.** (Sync handles notes-missing-from-`.bib` by adding them when
the user runs it; removal of `.bib` orphans is always a human decision.)

## Write the report

Write `<books_folder>/_Health.md`. It is **Full AI** — banner, no per-line `⚡`:

```bash
obsidian vault=<vault-name> create path="<books_folder>/_Health.md" content="<report>" silent
obsidian vault=<vault-name> property:set name="fullAI"  value="true"  path="<books_folder>/_Health.md"
obsidian vault=<vault-name> property:set name="mixedAI" value="false" path="<books_folder>/_Health.md"
```

The report has one `##` section per check above. Under each, list the offending notes as
wikilinks (`[[<title>]]`) with the specific problem, and end with a one-line summary count
(e.g. "3 notes missing required fields, 1 duplicate ISBN, 2 orphaned `.bib` entries"). The
first body line is the banner:

```
> [!ai] This health report is generated by the librarian skill.
```
